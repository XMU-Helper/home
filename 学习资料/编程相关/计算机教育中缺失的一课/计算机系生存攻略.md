# 生存手册
国内有很多高校的生存手册广为流传，我认为对在校生来说，不管什么时候阅读都会收获良多，但是本人还是希望能有一份更适合本校同学的手册可以参考。

因为本人在翔安学习了四年，并且还有疫情导致在家待了一个学期，后续封控限制等种种因素，让我们的大学体验受到局限，单凭一个人的力量很难完成一整份生存攻略。所以计划暂时从身边入手，将这本手册命名为计算机系生存手册，尝试整理一下记忆中的厦大。

可以写的内容：
1. 入学，高中到大学的转变
2. 对待课程，如何合理学习
3. 制定未来规划，读研，留学，就业，考公，选调等等
4. 生存技巧：转专业，选课抢课，课程安排，旁门左道
5. 毕业，毕业设计，答辩，校友会

## 培养方案
在这里列出计算机系某级本科生培养方案。

注意，因为大类招生，所以第一年是作为计算机类大类的学生学习的，所有信息学院的同学基本都是一样的安排。软件学院具体情况不清楚，因为不是同一个大类。

从上课体验来看，感觉部分课程是学院用于教学实践探索而开展的，所以变动应该会比较大，每一届的方案可能略有不同。以下内容仅供大家参考。

注：培养方案按照**课程名称，课程性质，学时，学分**的顺序列出。

### 第一学年
#### 第一学期
+ 大学语文，必修，32，2
+ C语言程序设计，必修，82，4
+ 体育课，必修，32，1
+ 微积分1-1，必修，96，5
+ 军事技能，必修，112，2
+ 线性代数（A），必修，64，3
+ 新生研讨课，必修，16，1
+ 思想道德修养与法律基础，必修，48，3
+ 大学英语（四），必修，64，2
+ 形势与政策（1），必修，8，0.25

注：
1. 厦大要求体育一共四学分，请大家合理在四年内修完，由于涉及到”选了体育课那么这一年的体测会在体育课上统一安排，而不是自己参加“的问题，所以请合理安排自己的体育课。
2. 军事技能指的是军训，两周时间，校内运动场军训。我们当时是入学后的两周，所以军训完直接国庆节放假。
3. 厦大有英语入学考试，按照考试成绩区分学习的级别，具体内容请看别的部分。简单来说，一二级要上两年共四个学期的课程，三四级是上一年，并且跳过的部分成绩记为3.7，如果没有记错的话。
4. 形势与政策一共要上四年，一学期一次，学期内一共4次课程，这门课应该是辅导员负责的。

### 总结
1. 课程总体安排、课程顺序安排还算合理。按照工科基础课程-语言课-核心课顺序。思政课、体育课、选修课等这里不谈。
2. 总体来说课程强度很大，大二课程量很大，大三看起来少一点，但是课程难度高。
